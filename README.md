# ScanAtlas

ScanAtlas is a medical studies search application focused on helping users quickly find, filter, and review relevant research.

## Overview

This repository is the starting point for the ScanAtlas project.

Core idea:
- Index medical studies metadata
- Provide fast and relevant search
- Surface key study details in a clear format

## Project Goals

- Make it easy to discover high-quality medical literature
- Reduce time spent reviewing large result sets
- Support future additions such as saved searches and ranking improvements
